# Horse Club - Landing Page

**Technologies:** HTML, SCSS, CSS, JavaScript, npm

## О проекте
Одностраничный сайт для демонстрации прайс-листа и услуг клуба.  
- Используется SCSS с переменными, миксинами и вложенностью  
- CSS генерируется из SCSS для подключения в браузере  
- Адаптивный дизайн для разных размеров экранов
